pyWinUSB
========

*A simple tool written in Python, made to help with creating a bootable
USB Windows drives* 
**Program support Windows Vista images and newer**

Installation:
-------------

::

    sudo pip install py-winusb

or using source code:

::

    sudo ./setup.py install

Running
-------

::

    pywinusb
